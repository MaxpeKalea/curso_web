~Etiquetas de Agrupación~

Son etiquetas que por defecto tienen un display bloque, pero no contribuyen al esquema de la página.

Párrafos

Se pueden crear párrafos con las etiquetas <p>, <pre> y <blockquote>.

Ejemplo de un párrafo preformateado (<pre>): 

Esto esta dentro
de la etiqueta pre

Se puede usar para fragmentos de código:

let x = 25
let y = 24
let z = x + y

La etiqueta blockquote se utiliza para crear citas literales en un bloque, tal y como hicimos al principio del curso.

La utilizamos para respetar los derechos de autor de la página original

Listas

Existen 3 tipos de lista:

  • Desordenada: <ul>
  • Ordenada: <ol>
  • De definiciones: <dl>

Una lista ordenada es la que por naturaleza tiene un orden (los días de la semana).

Ejemplo de lista ordenada:

  1. Lunes
  2. Martes
  3. Miércoles
  4. Jueves
  5. Viernes
  6. Sábado
  7. Domingo

Bloques neutros

La etiqueta <div> no tiene ningún valor semántico.

Unida al uso de las clases CSS es el principal elemento de diseño en la Web.

Bloques nuevos en HTML

  • <Address>
  • <Main>
  • <Figure>

Según aparece en la Web de MDN: El elemento HTML <address> aporta información de contacto para su <article> mas cercano o ancestro <body>; en el último caso lo aplica a todo el documento.

Normalmente puede ser colocado dentro de un elemento <footer> de la sección

El elemento HTML <Main> representa el contenido principal del <body> de un documento o aplicación. El área principal del contenido consiste en el contenido que está directamente relacionado, o se expande sobre el tema central de un documento o la funcionalidad central de una aplicación. Este contenido debe ser único al documento, excluyendo cualquier contenido que se repita a través de un conjunto de documentos como barras laterales, enlaces de navegación, información de derechos de autor, logos del sitio y formularios de búsqueda (a menos, claro, que la función principal del documento sea un formulario de búsqueda).

El elemento <Figure> es un contenedor de imágenes del que hablaremos al referirnos a éstas.

Otras etiquetas relacionadas con bloques

La etiqueta <hr> representa un cambio de tema entre párrafos, creando una línea horizontal.

La etiqueta <br> crea un salto de línea.

Estas etiquetas es recomendable usarlas lo menos posible, por su bajo valor semántico y porque pueden estropear el diseño.
