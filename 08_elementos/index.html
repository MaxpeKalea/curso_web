~Introducción~

¿Qué son las etiquetas HTML?

Las etiquetas HTML son el camino que se va haciendo para desarrollar una Web.

Todo lo que escribimos o subimos irá dentro de las etiquetas, que le dirán a la Web qué es lo que vamos haciendo.

Por un lado tendremos etiquetas para estructurar, que le dirán a la Web si lo que estamos escribiendo en su interior es la cabecera de nuestra página, es el cuerpo, es el pie de página o es un elemento aparte.

También tendremos etiquetas para indicar si nuestra intención es escribir un párrafo, subir una imagen, crear una lista, o para que la Web sepa qué palabras de nuestro texto son clave y nuestra página aparezca en el buscador cuando alguien trate de buscar contenido a través de esas palabras.

Las etiquetas han sido diseñadas con la forma de un texto rodeado por corchetes angulares  <etiqueta>

Pueden ser:

Sencillas (One-sided tag)

Por ejemplo, para indicar que queremos realizar un salto de línea podría usarse la etiqueta <br>

Dobles (Two-sided tag)

Donde  <etiqueta>  es la etiqueta de apertura y  </etiqueta>  es la etiqueta de cierre, indicada por la diagonal. Por ejemplo, para escribir un párrafo, primero indicamos con la etiqueta de apertura que el texto que va entre estas dos etiquetas es un párrafo, después escribimos nuestro párrafo dentro, y no nos olvidamos de cerrarlo con la etiqueta de cierre o nuestra página irá presentando errores. Ejemplo de párrafo bien estructurado:

<p>Caminante, no hay camino, se hace camino al etiquetar</p>
