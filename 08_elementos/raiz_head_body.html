Etiquetas Raíz

<!doctype html>

Con esta etiqueta simple indicamos que el documento está bajo el estándar HTML 5.

Se introduce como primera línea de nuestro documento HTML 5.

<html>

Se introduce como segunda línea de nuestro documento.

Representa la raíz de un documento HTML. El documento siempre debe cerrarse con la etiqueta de cierre </html>

Todo el contenido de nuestra Web estará dentro de esta etiqueta.

Etiqueta <Head>. Los Metadatos

La etiqueta <head> representa una colección de metadatos acerca del documento, incluyendo enlaces a, definiciones de, scripts y hojas de estilo.

Dentro de esta etiqueta se incluye el atributo "lang" para especificar el idioma de nuestro documento

Esta etiqueta alberga en su interior otras etiquetas:

<title>

Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada.

<meta>

Define los metadatos que no pueden ser definidos usando otro elemento HTML.

<link>

Se usa para enlazar JavaScript y CSS externos con el documento HTML actual. Tambien sirve para asignar un icono o favicon a nuestra web en la pestaña del navegador o en la lista de marcadores.

<style>

Etiqueta de estilo usada para escribir CSS en línea.
